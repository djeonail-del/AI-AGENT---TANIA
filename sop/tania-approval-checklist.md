# ✅ Tania Content Approval Checklist

_Used during Notion "Design Ready" review. Run through ALL sections before marking Approved or Rejected._

---

## 🔍 Pre-Review: Research First

Before looking at the design, verify the content:

1. Read the caption + headline created by Nara
2. **Web search** the topic from at least 3 different sources
3. Verify facts — numbers, dates, names, statistics, percentages
4. Check if the news is still relevant (not outdated / superseded)
5. Check the Source URL field in Notion — does it actually support the claim?
6. Note any corrections needed before opening the design

---

## 1. Akurasi (Accuracy)

- [ ] Fakta akurat dan terverifikasi dari multiple sources
- [ ] Berita masih relevan / tidak outdated (< 7 hari untuk breaking news, < 30 hari untuk tips)
- [ ] Tidak ada angka atau statistik yang salah
- [ ] Tidak ada nama orang/brand yang salah ketik
- [ ] Source URL valid dan mendukung klaim di konten
- [ ] Tidak ada klaim berlebihan / clickbait yang tidak bisa dibuktikan

---

## 2. Design (Visual)

- [ ] **Teks per elemen:** maks 1 kalimat / phrase singkat
- [ ] **Box cyan/biru:** BUKAN sebagai label kategori umum — hanya untuk highlight spesifik
- [ ] **Subjek/tokoh:** prominent, jelas, tidak terpotong
- [ ] **Orientasi objek:** natural (HP tidak kebalik, tangan kanan tidak jadi kiri, dll)
- [ ] **Background:** dark (hitam/gelap) sesuai brand
- [ ] **Font headline:** Ultra Condensed Bold — sesuai reference Ternakuang
- [ ] **Font di box cyan:** Inter SemiBold atau sejenis — BUKAN font yang sama dengan headline (kontras tipografi wajib)
- [ ] **AI artifacts:** tidak ada teks garbled, wajah deformed, objek floating/tidak natural yang mencolok
- [ ] **Branding:** logo AUTOFINT visible (tidak terpotong, tidak terlalu kecil)
- [ ] **Warna konsisten:** Cyan #0D7AB5/#16AAD8 + Orange #F97316

---

## 3. Caption

- [ ] **Hook** di kalimat pertama — langsung menarik perhatian
- [ ] Konten akurat, tidak misleading
- [ ] **Soft CTA** ke AUTOFINT (bukan hard sell; contoh: "Pantau keuanganmu di AUTOFINT")
- [ ] Tidak menjanjikan konten yang tidak ada di slide
- [ ] Tidak terlalu panjang (maks 150 kata untuk feed, 50 kata untuk Reel)
- [ ] Tidak ada typo atau grammatical error

---

## 4. Hashtag

- [ ] **Hanya 1 hashtag** — gunakan salah satu:
  - `#autofintnews` → untuk konten berita/informasi
  - `#autofinttips` → untuk konten tips/edukasi
- [ ] Tidak ada hashtag lain selain satu di atas

---

## 5. Jadwal (Schedule)

- [ ] Tanggal publish masuk akal (tidak di masa lalu, tidak terlalu jauh ke depan)
- [ ] Jam sesuai default:
  - **11:00 WITA** → konten berita
  - **15:00 WITA** → konten tips/edukasi
- [ ] Tidak conflict dengan post lain di hari yang sama

---

## 6. Action & Escalation

### Semua ✅ → Approve
```
Update Notion status → ✅ Design Approved
Notify Djeon via Telegram (always)
```

### Ada ❌ → Reject
```
Update Notion status → ❌ Rejected
Tulis feedback detail di field Notes:
  - Apa yang salah
  - Koreksi fakta yang diperlukan
  - Perubahan design/copy yang dibutuhkan
Notify Djeon via Telegram (always)
```

### Eskalasi ke Djeon
- Rejection **> 2x** untuk konten yang sama → langsung notify Djeon
- Konten sensitif (politik, kesehatan, keuangan major) → minta second opinion Djeon
- Tidak yakin dengan akurasi fakta setelah 3 sources → eskalasi
